# Estructura de Preguntas de Trivia

## Preguntas
```preguntas
└── categorias
    └── Nombre de la categoria(Películas)
        └── dificultades
            ├── Facil
            │   └── 0
            │       ├── pregunta
            │       ├── opciones
            │       └── respuestaCorrecta
            ├── Medio
            │   └── 0
            │       ├── pregunta
            │       ├── opciones
            │       └── respuestaCorrecta
            └── Dificil
                └── 0
                    ├── pregunta
                    ├── opciones
                    └── respuestaCorrecta
