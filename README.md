# Estructura de Preguntas de Trivia

## Preguntas

```preguntas
└── Películas
    ├── Facil
    │   └── 0
    │       ├── pregunta
    │       ├── opciones
    │       └── respuestaCorrecta
    ├── Medio
    │   └── 0
    │       ├── pregunta
    │       ├── opciones
    │       └── respuestaCorrecta
    └── Dificil
        └── 0
            ├── pregunta
            ├── opciones
            └── respuestaCorrecta ```
